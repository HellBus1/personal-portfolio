## Getting Started

1. **Clone the repository** :
```shell
git clone git@github.com:HellBus1/personal-portfolio.git
cd personal-portfolio
```

2. **Install dependencies** :
```shell
npm install
```

3. **Run the development server** :
```shell
npm run dev
```

4. **Build for production** :
```shell
npm run build
```
